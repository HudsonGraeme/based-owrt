Dependencies

Ultra-Minimal

Total bundle size: 44KB uncompressed, 10KB gzipped. No external dependencies, no CDN requests, no build tools required.

Modern UI

Clean, dark interface with glassmorphic cards, smooth animations, and responsive design. Built with vanilla JavaScript.

Complete Control

Manage network interfaces, wireless, firewall, DHCP, and system settings. Live stats, diagnostics, and quick actions.

Secure by Design

Uses OpenWrt's native ubus RPC API with session-based authentication and ACL-protected operations. Same security as LuCI.

Router-Optimized

Minimal CPU and memory footprint. Intelligent polling that pauses when tab is hidden. Hardware-accelerated animations.

Easy Deploy

One command to deploy. No installation, no configuration. Works alongside LuCI without conflicts.

Quick Start

# Deploy to your router
scp -r custom/* root@192.168.1.1:/www/custom/

# Access at http://192.168.1.1/custom/
