Based - Manual Test Checklist

Run these tests after deploying to verify functionality:

✓ Authentication

  • Navigate to http://router-ip/custom/
  • Login form should appear with WiFi icon logo
  • Enter credentials and click LOGIN
  • Should redirect to dashboard on success
  • Invalid credentials should show error message

✓ Dashboard

  • System stats should load (hostname, uptime, memory, load)
  • Memory progress bar should display
  • Load average graph should render (SVG)
  • CPU, network RX/TX stats should update
  • Stats should auto-refresh every 5 seconds
  • Quick action buttons (REBOOT, etc.) should be visible

✓ Network - Interfaces

  • Click NETWORK → INTERFACES tab
  • Interface table should load with LAN, loopback, etc.
  • Click "Configure" on an interface
  • Modal should open with current settings
  • Switch protocol between Static/DHCP
  • Static fields should show/hide appropriately
  • Cancel/close should dismiss modal

✓ Network - Diagnostics

  • Click DIAGNOSTICS tab
  • Enter 8.8.8.8 in ping field
  • Click RUN PING
  • Should show ping results with response times
  • Try traceroute with a hostname
  • Should display hop-by-hop route

✓ Network - DHCP

  • Click DHCP tab
  • Active leases table should populate
  • Should show hostname, IP, MAC, expiry
  • If no leases, should show "No active leases"

✓ System Settings

  • Click SYSTEM → GENERAL
  • Hostname field should be pre-populated
  • Timezone field should show current value
  • Click ADMINISTRATION tab
  • Password change form should be visible

✓ UI/UX

  • Dark theme with glassmorphic cards
  • Monospace font for data values
  • Smooth tab transitions and animations
  • Modal backdrop blur effect
  • Toast notifications appear/disappear smoothly
  • Responsive on mobile devices

✓ Session & Security

  • Session persists on page reload
  • Logout button clears session
  • After logout, redirects to login page
  • Expired session shows login page
  • No XSS vulnerabilities (input is escaped)

✓ Performance

  • Page loads in under 1 second on router
  • No console errors in browser DevTools
  • Network tab shows all resources load successfully
  • Polling stops when tab is hidden
  • Minimal memory usage (check Task Manager)

Note: This is a manual test checklist. For automated tests, consider adding Playwright or Cypress tests in the future.
